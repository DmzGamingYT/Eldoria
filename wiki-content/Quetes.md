## Détails walkthrough

### Q1 — Chasse aux Slimes
- **Donneur** : Aldric l'Ancien (Village central).
- **Objectif** : Tuer 5 Slimes Verts dans les Plaines verdoyantes.
- **Récompense** : 50 XP, 30 po, Potion de Soin.
- **Astuce** : Stay close to the spawn, pop Bouclier Arcanique pour les retours d'ATQ Slime (très lent).

### Q2 — La Menace Gobeline
- **Donneur** : Brynn la Marchande.
- **Objectif** : Tuer 6 Pillards Gobelins dans la Forêt des gobelins.
- **Récompense** : 120 XP, 80 po, Épée de Fer.
- **Astuce** : Les gobelins fuient quand HP < 30% — finissez-les avec Boule de Feu à distance.

### Q3 — Chasse aux Loups
- **Donneur** : Saela la Chasseuse (Bois sinistres).
- **Objectif** : Tuer 5 Loups Sinistres.
- **Récompense** : 180 XP, 100 po, Cotte de Mailles.
- **Astuce** : Cycle crépuscule = +30% spawn loups. Quarnez-les vers les arbres pour les isoler.

### Q4 — Repos des Os
- **Donneur** : Mireille la Sage.
- **Objectif** : Tuer 6 Guerriers Squelettes (Cimetière ancien).
- **Récompense** : 300 XP, 200 po, Hache d'Acier.
- **Astuce** : Capstone Combat débloquée maintenant (`Bourreau` crit ×2.5) — privilégiez les talents Combat sur cette quête.

### Q5 — Le Seigneur des Ombres
- **Donneur** : Mireille la Sage (auto-triggered post-Q4).
- **Objectif** : Vaincre **Mordrak** dans le Donjon (zone nord).
- **Récompense** : 1000 XP, 1000 po, **Tueuse de Dragon** (légendaire, +ATQ ×2.5).
- **Astuce** : Avoir les 3 capstones OU privilégier Magie (regen mana) OU Combat (crit tank). Voir [Wiki / Arbre-de-Talents](Arbre-de-Talents).
