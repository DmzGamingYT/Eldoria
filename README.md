## ⚔️ Eldoria en bref

**Eldoria** est un RPG 3D fantasy action — jouable dans le navigateur ou installable sur Windows, macOS et Linux. Monde procédural 200×200, combat en temps réel, 5 sorts, **arbre de talents à 3 branches**, 5 quêtes chaînées jusqu'au boss final **Mordrak**. 🆕 v0.3.0.

> 📥 [Télécharger v0.3.0](#-téléchargements) · 📖 [Wiki complet](https://github.com/DmzGamingYT/Eldoria/wiki) · 🚀 [Démarrer en local](docs/dev-setup.md) · 🤝 [Contribuer](CONTRIBUTING.md)

## 📥 Téléchargements

> **Dernière version stable : [v0.3.0 ▸ Releases](https://github.com/DmzGamingYT/Eldoria/releases/latest)** — installeurs natifs générés par la CI à chaque tag `v*`.

| Plateforme | Format | Installation |
|:--:|:--|:--|
| 🪟 **Windows** 10 / 11 | NSIS `Eldoria-0.3.0-win-x64.exe` (~265 Mo) | Double-clic |
| 🍎 **macOS** Intel | DMG + ZIP (`Eldoria-0.3.0-mac-x64.*`, ~350 Mo) | Glisser dans `/Applications` |
| 🍎 **macOS** Apple Silicon | DMG + ZIP (`Eldoria-0.3.0-mac-arm64.*`, ~348 Mo) | Glisser dans `/Applications` |
| 🐧 **Linux** universal | AppImage `Eldoria-0.3.0-linux-x64.AppImage` (~410 Mo) | `chmod +x` puis double-clic |
| 🐧 **Debian / Ubuntu / Mint** | `.deb` `eldoria_0.3.0_amd64.deb` (~298 Mo) | `sudo dpkg -i …deb` |
| 🐧 **Fedora / RHEL / Nobara** | `.rpm` `Eldoria-0.3.0-linux-x64.rpm` (~230 Mo) | `sudo rpm -i …rpm` |

**Liens directs :**

| Plateforme | Lien |
|:--|:--|
| 🪟 Windows | [`.exe`](https://github.com/DmzGamingYT/Eldoria/releases/latest/download/Eldoria-0.3.0-win-x64.exe) |
| 🍎 macOS Intel (DMG) | [`.dmg`](https://github.com/DmzGamingYT/Eldoria/releases/latest/download/Eldoria-0.3.0-mac-x64.dmg) · [`.zip`](https://github.com/DmzGamingYT/Eldoria/releases/latest/download/Eldoria-0.3.0-mac-x64.zip) |
| 🍎 macOS Apple Silicon (DMG) | [`.dmg`](https://github.com/DmzGamingYT/Eldoria/releases/latest/download/Eldoria-0.3.0-mac-arm64.dmg) · [`.zip`](https://github.com/DmzGamingYT/Eldoria/releases/latest/download/Eldoria-0.3.0-mac-arm64.zip) |
| 🐧 Linux | [`AppImage`](https://github.com/DmzGamingYT/Eldoria/releases/latest/download/Eldoria-0.3.0-linux-x64.AppImage) · [`.deb`](https://github.com/DmzGamingYT/Eldoria/releases/latest/download/eldoria_0.3.0_amd64.deb) · [`.rpm`](https://github.com/DmzGamingYT/Eldoria/releases/latest/download/Eldoria-0.3.0-linux-x64.rpm) |

```bash
# Linux AppImage (universelle)
wget https://github.com/DmzGamingYT/Eldoria/releases/latest/download/Eldoria-0.3.0-linux-x64.AppImage
chmod +x Eldoria-0.3.0-linux-x64.AppImage && ./Eldoria-0.3.0-linux-x64.AppImage

# Debian / Ubuntu
sudo dpkg -i eldoria_0.3.0_amd64.deb
```

> ⚠️ **Installeurs non signés** (normal pour un indie game) — voir [`docs/dev-setup.md`](docs/dev-setup.md) pour le dépannage Gatekeeper (`xattr -cr`).

## 🤝 Contribuer

Les contributions sont les bienvenues — voir [`CONTRIBUTING.md`](CONTRIBUTING.md). PR ouvertes contre `main` avec checks verts (Lint + Typecheck + Build).

```bash
bun run lint            # ESLint
bunx tsc --noEmit       # Typecheck
bun run build           # Build de production
```

Pour un changement substantiel, ouvrez d'abord une _issue_ avec votre approche.
