## ⚔️ Qu'est-ce qu'Eldoria ?

**Eldoria** est un RPG 3D fantasy action open-source. Vous incarnez un voyageur appelé par **Aldric l'Ancien** pour repousser les sbires de **Mordrak, le Seigneur des Ombres**.

| | |
|:--|:--|
| 🌍 **Monde** | Procédural 200×200 · cycle jour/nuit 180 s · **7 biomes** dont Frostpeak |
| ⚔️ **Combat** | Temps réel · arc d'attaque + combo + iframe · **5 sorts** sur barre rapide |
| 🎭 **Progression** | **18 talents** en 3 branches (Combat / Magie / Survie) · multi-rang |
| 🎒 **Butin** | **18+ objets** en 5 raretés · 3 sets légendaires · boutique + craft |
| 🧩 **Quêtes** | **6 quêtes** chaînées → boss final **Mordrak** dans une arène dédiée |
| 🖥️ **Stack** | Next.js 16 · R3F 9 · Three.js · Zustand · TypeScript · Electron 42 |

> 📖 [Wiki complet](https://github.com/DmzGamingYT/Eldoria/wiki) · 📋 [Changelog](CHANGELOG.md) · 🏗️ [Architecture](docs/development/architecture.md)

## 📥 Téléchargements

> **[v0.4.0 ▸ Releases](https://github.com/DmzGamingYT/Eldoria/releases/latest)** — installeurs natifs Windows / macOS / Linux.

| Plateforme | Format | Installation |
|:--:|:--|:--|
| 🪟 **Windows** 10/11 | NSIS `.exe` *(~265 Mo)* | Double-clic |
| 🍎 **macOS** Intel + Apple Silicon | DMG + ZIP *(~350 Mo)* | Glisser dans `/Applications` |
| 🐧 **Linux** universal | AppImage *(~410 Mo)* | `chmod +x` puis double-clic |
| 🐧 **Debian / Fedora** | `.deb` / `.rpm` *(~230-298 Mo)* | `sudo dpkg -i` / `sudo rpm -i` |

> 🎮 **[itch.io](https://dmzgamingyt.itch.io/eldoria)** — jouez directement dans votre navigateur, zéro installation.
> ⚠️ Installeurs non signés par défaut — guide de signature Apple : [`docs/release/apple-signing-guide.md`](docs/release/apple-signing-guide.md).

## 🤝 Contribuer

Recherchez les issues **[`invite-to-collaborate`](https://github.com/DmzGamingYT/Eldoria/issues?q=is%3Aissue+label%3Ainvite-to-collaborate)** 🎯 — des PRs cadrés pour bien débuter.

```bash
bun install           # installe les dépendances
bun run lint          # ESLint
bunx tsc --noEmit     # typecheck TypeScript
bun run build         # build de production Next.js
bun run electron:dev  # lance le jeu en mode desktop (Electron)
```

Conventions du projet : [`CONTRIBUTING.md`](CONTRIBUTING.md). Pour un changement substantiel, ouvrez d'abord une _issue_ avec votre approche.
